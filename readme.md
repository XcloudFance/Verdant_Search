# verdant search
目前为了考虑后端的负载量，将搜索引擎的后端改成了flask
但是其临时存储结构CubeQL用了fastapi

## Key Feature

- 搜索引擎会自动从其他搜索引擎上寻找数据库缺失的搜索结果，自动更新
- 直接关联有道词典，查词更方便
- 搜索没有广告，界面更加简洁
- 无痕搜索，不会产生大数据杀熟
- 对csdn等博客网页实现去相似化，提高了搜索效率(coming)

## CubeQL

用于青荇搜索的临时存储结构，实现类似redis的功能，同时还能通过布隆过滤器模块来过滤已经爬虫过的网址，用作去重
